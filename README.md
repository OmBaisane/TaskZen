🚀 TaskZen – Task Manager App

A modern Task Management Web App built using React + TypeScript, designed to help users manage daily tasks efficiently with a clean UI and smooth user experience.

---

🌐 Live Demo

👉 https://task-zen-omega.vercel.app/

---

📌 Features

✅ Add, Edit, Delete Tasks (Full CRUD)
✅ Mark Tasks as Completed / Pending
✅ Filter Tasks (All / Pending / Done)
✅ 🌙 Dark Mode / ☀️ Light Mode Toggle
✅ 💾 Data Persistence using LocalStorage
✅ ⚡ Fast and Responsive UI
✅ 🧩 Component-Based Architecture

---

🛠️ Tech Stack

- ⚛️ React
- 🔷 TypeScript
- 🎨 Tailwind CSS
- 🌐 Vercel (Deployment)

⚙️ Installation & Setup

1. Clone the repository:

git clone https://github.com/OmBaisane/TaskZen

2. Navigate to project folder:

cd task-zen

3. Install dependencies:

npm install

4. Start development server:

npm start
