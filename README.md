# TaskZen 🚀

A Task Manager App built with React + TypeScript.

## Features

- Add / Delete / Edit Tasks
- Mark tasks as Done
- Filter (All / Pending / Done)
- Dark Mode 🌙
- LocalStorage (data saved)

## Tech Stack

- React
- TypeScript
- Tailwind CSS
